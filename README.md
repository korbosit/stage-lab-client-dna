## 🎯 Что внутри

Документ отвечает на главный вопрос: **кто на самом деле покупает у Stage Lab и почему.**

- **Идентичность и мировоззрение ЦА** — демография, ценности, скрытые амбиции
- **Боли, страхи и триггеры** — 5 ключевых проблем, топ-5 глубинных страхов, реальные болевые ситуации
- **Барьеры, мифы и прошлый негативный опыт** — что мешает купить прямо сейчас
- **Трансформация и возражения рынка** — что клиент получает и чего боится на старте
- **Триггеры покупки** — что именно закрывает продажу в момент решения
- **Итоговый аватар клиента** — «Осознанная мама на низком старте»
- **Рекомендации для стратегии** — как использовать выводы в Deep Research, оффере и контенте

## ⚡ Снимок аудитории

| | |
|---|---|
| **Кто** | Мама 32–48 лет (в 8 из 10 случаев — главный инициатор покупки) |
| **Ребёнок** | 4–16 лет, пиковые сегменты 5–9 и 10–14 лет |
| **Гео** | Русскоязычные и международные семьи в США + весь мир онлайн |
| **Доход** | $150,000–$500,000+/год, семейный |
| **Идентичность** | «Я — мама, которая не упускает возможности» |
| **Главный страх** | «Я испортила ребёнку детство своими амбициями» |
| **Что закрывает продажу** | Реальный кейс успеха + личный разговор с основателем, которая сама прошла этот путь как мама |

## 🧭 Как использовать этот документ

1. **Для стратегии и позиционирования** — раздел «Рекомендации для Deep Research и маркетинговой стратегии» в конце документа.
2. **Для текстов и офферов** — прямые цитаты клиентов из разделов «Боли, страхи и триггеры» и «Триггеры покупки» готовы к использованию как заголовки, подзаголовки и возражения в FAQ.
3. **Для брифинга команды/подрядчиков** — раздел «Итоговый аватар клиента» даёт сжатый портрет за 30 секунд чтения.

## 🛠 Как включить веб-версию (GitHub Pages)

Чтобы ссылка на оформленную версию заработала:

1. Открыть репозиторий на GitHub → **Settings** → **Pages**
2. В блоке **Build and deployment** → **Source** выбрать `Deploy from a branch`
3. **Branch**: `main`, папка **`/docs`** → **Save**
4. Через 1–2 минуты документ будет доступен по адресу `https://korbosit.github.io/stage-lab-client-dna/`
